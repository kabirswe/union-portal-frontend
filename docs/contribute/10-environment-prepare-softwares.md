# Environment - Prepare Softwares

## Git

* **Must set `git config core.ignorecase false`**


## Node.js

* Use latest LTS version (v14.xx.xx)

We recommend https://github.com/tj/n to manage your Node.js versions.


## Yarn

* Use latest version `^1.22.10`
* We'll use many npm-scripts with options and arguments. `yarn` is suitable for such a case.


## Text editor

We recommend VSCode but you can use your favorite one which has features below.

* TypeScript integration (prediction, code jump, showing type errors, etc...)
* ESLint integration (show warnings of TypeScript coding rules)
* SCSS integration
* Stylelint integration (show warnings of SCSS coding rules)
* Prettier integration (show warnings of formatting, auto fix)
